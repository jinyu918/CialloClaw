# 缺失部分

## 多插件和多模型

支持多生态插件、多模型配置 

通过插件化“感知包”扩展不同场景能力，让它既能服务开发者，也能服务日常办公和娱乐环境。

内置兼容社区 Skills 生态，Agent 可以从 GitHub 安装技能。

可以切换不同大模型，不只是OpenAI的GPT
